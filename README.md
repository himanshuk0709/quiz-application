# Python Quiz Application

This is a beginner-friendly Python project that implements a terminal-based quiz application.
The project was built to practice program flow, conditional logic, and user interaction.

---

## Features

- Displays multiple-choice questions
- Accepts user input for answers
- Checks answers and updates score
- Displays the final score after the quiz
- Simple menu to start or exit the quiz

---

## Concepts Used

- Python dictionaries
- Loops and conditional statements
- Control flow and state tracking
- User input handling
- Formatted output using f-strings

---

## How the Quiz Works

1. The user starts the quiz from the menu
2. Questions are displayed one by one with multiple-choice options
3. The user enters an answer (A / B / C / D)
4. The program checks the answer and updates the score
5. After all questions are answered, the final score is displayed
